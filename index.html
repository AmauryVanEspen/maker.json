What is civic.json?

Civic.json is a metadata file (and standard) that helps people learn more about your community-centric code projects!

By putting one in your project's GitHub repository, you can tell others what it's about, its status, which people and organizations are involved, and whom they can reach out to for more information.

How do I create one?

The Builder Form is the easiest way to start your civic.json file.

You can also use the Validation Tool to check that your repository's civic.json file is correctly formatted.

Who is behind this?

Civic.json was originally created by Beta.NYC. We here in DC—members of the Code for DC brigade and employees of the DC Government—have extended their specification to make it easier for local governments and civic technologists collaborate on projects.
