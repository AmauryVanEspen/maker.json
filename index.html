What is civic.json?

Civic.json is a metadata file (and standard) that helps people learn about your community-centric code projects.

By putting one in your project's GitHub repository, you can tell others what it's about, its status, which people and organizations are involved, and whom they can reach out to for more information.

Think of it like a structured "README" file that makes possible things like browseable project pages for your organization or city.

Want to see an example? Here's one for the civic.json specification, itself!

How do I create one?

The Civic.json Builder is the easiest way to start your civic.json file.

With the Civic.json Validator you can also check if an existing civic.json file is correctly formatted.

Who is behind this?

Civic.json was originally created by Beta.NYC in 2014.

Now, a group of Washington, DC residents—Code for DC volunteers and employees of the District Government—have extended their specification to make it useful for a broader user base.

In doing so, we aim to foster collaboration and reuse across government agencies, municipalities, and the citizen-government divide.
